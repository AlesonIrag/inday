## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (usually http://localhost:5173)

## Customization

### Love Letter
Edit `src/components/LoveLetter.jsx`:
- Change the `pages` array to customize messages
- Modify the `reasons` array to add your own reasons

### Photo Gallery
Add your photos to `public/photos/` folder and update the photo array in `src/components/PhotoGallery.jsx`

### Memory Timeline
Edit `src/components/MemoryTimeline.jsx` to add your special moments

### Love Quiz
Customize questions in `src/components/LoveQuiz.jsx`

### Music Player
Update the playlist in `src/components/MusicPlayer.jsx` with your favorite songs
